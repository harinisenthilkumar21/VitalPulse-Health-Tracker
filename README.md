# VitalPulse Health Dashboard

A modern, professional health monitoring and tracking SaaS application.

## 🚀 Features

- **SaaS Dashboard**: Comprehensive overview of health score, status, and streaks.
- **Vitals Tracking**: Monitor Heart Rate, Blood Pressure, Temperature, and SpO2.
- **Lifestyle Monitoring**: Track steps, sleep, water intake, and calories.
- **Multi-step Registration**: Smooth onboarding process.
- **Insights & Trends**: Visual representation of health data over time.
- **Alert System**: Real-time highlighting of abnormal values.
- **Responsive Design**: Works perfectly on desktop and mobile.

## 🛠️ Tech Stack

- **Frontend**: React.js, Tailwind CSS
- **Icons**: Lucide React
- **Animations**: Motion (formerly Framer Motion)
- **Charts**: Recharts

## 📂 Folder Structure

- `src/components`: Reusable UI components and layout elements.
- `src/pages`: Individual page components (Dashboard, Insights, etc.).
- `src/types.ts`: TypeScript interfaces and types.
- `src/mockData.ts`: Sample data for demonstration.
- `src/App.tsx`: Main application logic and routing.

## 🚦 Getting Started

1. **Install Node.js**: Ensure you have Node.js installed on your machine.
2. **Install Dependencies**:
   ```bash
   npm install
   ```
3. **Setup Environment Variables**:
   - Copy `.env.example` to `.env`
   - Update the variables as needed.
4. **Start the Project**:
   ```bash
   npm run dev
   ```

## 📝 Environment Variables

- `VITE_API_URL`: The base URL for the backend API.
- `VITE_APP_NAME`: The name of the application.
